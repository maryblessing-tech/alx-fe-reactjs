# ALX Frontend React Projects

This repository contains React projects completed as part of the ALX Frontend Engineering program.

## Projects

### 1. Tailwind React Integration

A React application demonstrating the integration and usage of Tailwind CSS for styling components.

**Location:** `tailwind-react-integration/`

**Technologies Used:**
- React
- Vite
- Tailwind CSS

**Features:**
- Tailwind CSS configuration with Vite
- Utility-first CSS styling
- Responsive design
- Interactive components with hover effects

### Setup Instructions

1. Navigate to the project directory
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
